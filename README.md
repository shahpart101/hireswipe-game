# HireSwipe Game

A founder-oriented and viral hiring simulator built with React and Vite.

## About
Swipe between two candidates like Tinder…
Get an ELO score…
Save results to leaderboard…

## Features
✔ Tinder-style swipe decisions  
✔ Hiring ELO scoring  
✔ Inline leaderboard  
✔ Shareable results

## How to Run
1. npm install  
2. npm run dev

## Tech Used
React, Vite, HTML2Canvas, CSS.
